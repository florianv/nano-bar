# < nano-bar >

> [Nanobar](http://nanobar.micronube.com) as a Polymer element

## Installation

```bash
$ bower install nano-bar --save
```

## Import

Import Web Components polyfill:

```html
<script src="../platform/platform.js"></script>
```

Import the element:

```html
<link rel="import" href="../nano-bar/nano-bar.html">
```

## Usage

```html
<nano-bar></nano-bar>
```
